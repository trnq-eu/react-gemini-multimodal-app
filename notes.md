`nvm install node`

install google ai:
`npm install @google/generative-ai`

upgrade es lint:
`npm install eslint@latest --save-dev`

downgrade es lint
`npm uninstall eslint`

`npm run start:frontend`

`npm run start:backend`

`npm install`
